# navigator.nvim

Floating tree file explorer for Neovim with file preview.

Keybinds:
  - `<leader>t` - opens the file explorer
  - `<CR>` — open file / expand folder  
  - `<Esc>` — close the window

### Lazy.nvim
```lua
{
  "Cpoing/navigator.nvim",
}
```
